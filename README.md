# vibe-check-dc604

The two take-home "assessment" repos from **Vibe Check: Exploiting Trust —
from Developers to AI agents** (DC604, June 2026).

These are exactly what a candidate is handed in the talk's fake-recruiter
scenario. Published so you can read through them at your own pace.

> ⚠️ **Read these in a sandbox — don't run them on your real machine.**
> Treat any unfamiliar repo as hostile until you've looked it over; that
> habit *is* the lesson. These copies are defanged for the workshop, but
> read them like the real thing.

## Contents

- **`assessment-1/`** — backend take-home: a Node/Express auth service with a
  login bug to fix. See its [README](assessment-1/README.md).
- **`assessment-2/`** — frontend take-home: a small Node utility that crashes
  on bad input. See its [README](assessment-2/README.md).

Each folder reads like an ordinary technical screen. Whether that's all it
is, is left for you to find out — the full walkthrough is in the talk.
